# AgileInit — Strategic Strategic Engine (V2)

## Description
AgileInit est une plateforme modulaire pour le cadrage stratégique de projets techniques. 
Elle permet aux Product Owners (PO) et aux Développeurs de transformer une idée brute en un pack de documentation et de prompts Antigravity exploitables en moins de 10 minutes.

## Fonctionnalités
- 🛠 **Workshop Cadrage** : Définition des personas et rédaction de User Stories avec suggestions intelligentes.
- 📑 **Auto-Docs Factory** : Génération automatisée de 7 fichiers Markdown standards (Vision, US, Backlog, Archi, Plan, Prompts, README).
- ⚡ **Antigravity Helper** : Vue optimisée pour les agents de codage avec segments de prompts prêts à copier.
- 🎨 **Design Premium** : Interface "Kinetic Deep-Space" avec mode dark, glassmorphism et transitions fluides (via Stitch MCP).

## Structure du Projet (Screaming Architecture)
```txt
src/
├── project-init/           # Cadrage initial (Personas, US)
├── doc-generator/          # Génération automatique (Markdown Factory)
├── generate-docs/          # Services d'export et templates backend
├── shared/                 # Store et utilitaires transverses
└── ui-agile-form/          # Interface React Premium
```

## Démarrage Rapide
1. Installer les dépendances : `npm install`
2. Lancer l'application : `npm run dev`
3. Ouvrir le navigateur et choisir entre le mode **Workshop** ou **Auto-Docs**.
