# Vision Produit — AgileInit

## Problème
Les juniors (dev, PO, étudiants) peinent à cadrer rapidement un projet Agile.
Le cadrage initial est long, mal structuré et rarement documenté proprement
(US, personas, objectifs, backlog) [web:54][web:58].

## Solution
Une web app guidée qui :
- Pose des questions simples (objectif, contexte, utilisateurs).
- Aide à rédiger des US propres via des placeholders.
- Génère automatiquement les fichiers Markdown (besoins, backlog, stack).
- Offre une UI soignée grâce à Stitch + Antigravity.

## Utilisateurs cibles
- Étudiants en dev / PO.
- Juniors en ESN.
- Freelances qui veulent cadrer vite un mini projet client.

## Objectifs SMART
- Cadrer 3 projets tests complets (US, backlog, exports) en <15min chacun.
- Obtenir au moins 5 retours d’utilisateurs, dont 3 estiment que l’outil
  remplace leur process actuel.
- Générer des exports Markdown utilisables sans retouche pour 80 % des US.

## Non-objectifs (v1)
- Pas de gestion multi-projet avancée.
- Pas d’intégration directe Jira/Trello.
- Pas de moteur d’IA complexe pour la suggestion (logique simple v1).
